# 任务 1：微调在 ImageNet 上预训练的卷积神经网络实现宠物识别

## 一、任务概述

本任务的目标是在 Oxford-IIIT Pet Dataset 数据集上完成宠物品种识别实验。该任务属于细粒度图像分类问题，模型需要根据宠物图像判断其所属品种。相比普通图像分类任务，宠物品种识别对模型的局部纹理、毛色、脸部结构、耳朵形态和身体轮廓等细节特征提出了更高要求。

本实验以 ResNet-18 作为基础卷积神经网络结构，使用在 ImageNet 数据集上预训练得到的权重初始化模型主体，并将原始分类层替换为适用于 Oxford-IIIT Pet Dataset 的 37 类输出层。训练过程中，新分类层从随机初始化开始学习，而卷积主干网络使用较小学习率进行微调。随后，本文进一步进行了超参数搜索、预训练消融实验以及基于 SE-block 的注意力机制实验，以分析不同训练设置和模型结构对分类准确率的影响。

## 二、数据集与预处理

本实验使用 Oxford-IIIT Pet Dataset。该数据集包含 37 个宠物类别，包括不同品种的猫和狗。实验中将任务设置为 37 类图像分类任务，模型输入为宠物图像，输出为对应的类别标签。

数据集划分方面，使用官方提供的 `trainval` 和 `test` 两个部分。其中，`trainval` 进一步按照 8:2 的比例划分为训练集和验证集，官方 `test` 部分作为最终测试集。为了保证实验可复现，划分过程中固定随机种子为 42。

图像预处理方面，所有图像被统一调整为 224×224，以适配 ResNet-18 的标准输入大小。训练集使用随机水平翻转、随机旋转和颜色扰动等数据增强方式，以提高模型泛化能力。验证集和测试集不使用随机增强，仅进行尺寸调整、张量转换和 ImageNet 标准均值方差归一化。

数据加载测试结果如下：

```text
Number of classes: 37
Train batch shape: torch.Size([8, 3, 224, 224])
Label batch shape: torch.Size([8])
```

该结果说明数据集类别数、图像尺寸和标签格式均符合后续训练要求。

## 三、Baseline 方法设计

### 3.1 模型结构

Baseline 模型采用 ResNet-18。原始 ResNet-18 是在 ImageNet 上训练的 1000 类分类模型，而本任务需要完成 37 类宠物品种识别，因此需要将最后的全连接分类层替换为新的输出层：

```python
in_features = model.fc.in_features
model.fc = nn.Linear(in_features, 37)
```

这样，ResNet-18 的卷积主干仍然可以继承 ImageNet 预训练特征，而最后的分类层则从随机初始化开始学习新的宠物分类任务。

### 3.2 分层学习率微调策略

为了满足“从零开始训练新的输出层，并对其余参数使用较小学习率进行微调”的要求，实验中对模型参数进行了分组。卷积主干网络使用较小学习率，分类层使用较大学习率。其原因是卷积主干已经从 ImageNet 中学习到了较通用的视觉特征，不应在新任务上被过快破坏；而新分类层是随机初始化的，需要更大的学习率快速适应 37 类宠物分类任务。

在最终主要实验中，采用如下设置：

```text
Backbone learning rate: 0.0001
Classifier learning rate: 0.001
Optimizer: AdamW
Weight decay: 0.0001
Batch size: 32
Image size: 224
```

## 四、实验工程实现

为了方便实验管理，本任务将项目划分为多个模块。`datasets.py` 负责数据集加载与预处理，`models.py` 负责构建 Baseline 模型和参数分组，`engine.py` 负责单轮训练与验证，`train.py` 负责完整训练流程，`test.py` 负责加载最优 checkpoint 并在测试集上评估，`plot_history.py` 负责绘制训练曲线，`grid_search.py` 用于批量搜索超参数组合。

实验过程中，每次训练都会保存验证集准确率最高的模型 checkpoint，并记录训练过程中的 loss 和 accuracy。测试阶段使用验证集上表现最好的 checkpoint，而不是简单使用最后一个 epoch 的模型，这样可以降低过拟合对最终测试结果的影响。

在超参数分析阶段，本文以 ImageNet 预训练的 ResNet-18 作为基础模型，固定模型结构、数据划分、图像尺寸、优化器和数据增强策略，主要观察不同学习率组合对模型性能的影响。实验中重点调整主干网络学习率和分类层学习率。

```text
Epochs: 15 或 75

Batch size: 32
Optimizer: AdamW
Weight decay: 0.0001
```

其中，15 epoch 用于超参数选择阶段，75 epoch 用于后续更充分的预训练消融和注意力机制对比实验。

## 六、预训练消融实验

### 6.1 实验目的

为了分析 ImageNet 预训练对宠物识别任务的影响，本文进行了预训练消融实验。实验中使用完全相同的 ResNet-18 结构、数据划分、训练轮数、学习率、优化器和 batch size。唯一不同之处是：一个模型使用 ImageNet 预训练权重初始化，另一个模型从随机初始化开始训练。

这样可以较公平地观察预训练权重对模型性能和泛化能力的影响。

### 6.2 实验结果

| 模型 | 是否使用 ImageNet 预训练 | Epochs | Best Val Acc | Test Loss | Test Acc |
|---|---|---:|---:|---:|---:|
| ResNet-18 | 是 | 75 | 0.9334 | 0.5378 | 0.8724 |
| ResNet-18 | 否 | 75 | 0.4959 | 2.6626 | 0.4424 |

实验结果表明，使用 ImageNet 预训练的 ResNet-18 在验证集和测试集上都显著优于随机初始化模型。在相同训练 75 个 epoch 的情况下，预训练模型的测试准确率达到 87.24%，而随机初始化模型只有 44.24%。这说明对于 Oxford-IIIT Pet Dataset 这种规模相对有限的细粒度分类任务，预训练权重具有非常重要的作用。

从结果可以看出，ImageNet 预训练不仅能够加快模型收敛，还能显著提升最终泛化性能。由于 ImageNet 预训练模型已经学习到边缘、纹理、颜色、形状和局部结构等通用视觉特征，因此在迁移到宠物品种分类任务时，只需要对这些特征进行微调，就能较快适应新任务。相比之下，从随机初始化开始训练的模型需要在较小数据集上同时学习低层视觉特征和高层语义特征，因此训练难度更大，最终性能明显较低。

值得注意的是，在超参数搜索阶段，15 epoch 的预训练模型测试准确率约为 88.20%，而 75 epoch 的预训练模型测试准确率为 87.24%，并没有进一步提升。这说明单纯延长训练轮数不一定带来更好的测试集表现，模型可能在训练集或验证划分上产生一定过拟合。因此，训练轮数需要结合验证集表现和测试集泛化能力综合选择。

## 七、注意力机制实验

### 7.1 SE-block 原理

为了进一步改进 Baseline 模型，本文在 ResNet-18 的基础上手动加入了 SE-block，也就是 Squeeze-and-Excitation block。SE-block 是一种通道注意力机制，其核心思想是让网络自动学习不同通道特征的重要性。

在普通卷积神经网络中，某一层输出的特征图通常具有形状 \(C \times H \times W\)，其中 \(C\) 表示通道数，\(H\) 和 \(W\) 表示空间尺寸。不同通道往往对应不同类型的视觉特征，例如边缘、纹理、颜色、局部形状或更高层语义模式。普通卷积网络虽然会利用这些通道，但不会显式判断哪些通道对当前输入更重要。

SE-block 首先通过全局平均池化将每个通道的空间特征压缩为一个标量，这一步称为 Squeeze。它相当于提取每个通道在整张图像上的全局响应。随后，SE-block 使用两层全连接网络学习通道之间的依赖关系，并通过 Sigmoid 函数生成 0 到 1 之间的通道权重，这一步称为 Excitation。最后，这些权重会乘回原始特征图，从而增强重要通道、抑制不重要通道。

对于宠物品种识别任务，SE-block 具有一定合理性。不同宠物品种之间的差异可能非常细微，例如毛发纹理、脸部结构、耳朵形状和颜色分布等。通道注意力机制可以帮助模型更关注对分类有用的特征通道，从而有机会提升细粒度分类性能。

### 7.2 SE-block 插入方式

本文没有直接替换整个模型，而是在 ResNet-18 的 BasicBlock 中加入 SE-block。具体来说，SE-block 被放置在第二个 BatchNorm 之后、残差连接相加之前。原始 BasicBlock 的结构可以简化表示为：

```text
conv1 -> bn1 -> relu -> conv2 -> bn2 -> residual add -> relu
```

加入 SE-block 后变为：

```text
conv1 -> bn1 -> relu -> conv2 -> bn2 -> SE-block -> residual add -> relu
```

这样做的好处是，SE-block 可以先对残差分支产生的特征进行通道重标定，然后再与 shortcut 分支相加。由于引入 SE-block 后模型结构已经不同于 torchvision 原始 ResNet-18，因此加载预训练权重时只加载名称和形状都匹配的参数。原有卷积层和 BatchNorm 层可以继承 ImageNet 预训练参数，而新增的 SE-block 参数和新的分类层从随机初始化开始训练。

### 7.3 注意力机制实验结果

| 模型 | 是否预训练 | 注意力机制 | Epochs | Best Val Acc | Test Loss | Test Acc |
|---|---|---|---:|---:|---:|---:|
| ResNet-18 | 是 | 无 | 75 | 0.9334 | 0.5378 | 0.8724 |
| ResNet-18 + SE-block | 是 | SE | 75 | 0.9063 | 0.6080 | 0.8621 |

实验结果显示，加入 SE-block 后，模型测试准确率为 86.21%，略低于原始 ResNet-18 Baseline 的 87.24%。这说明在当前实验设置下，SE-block 并没有带来性能提升。

这一结果并不意味着注意力机制无效，而是说明在当前数据规模、训练策略和模型设置下，SE-block 的增益有限。可能原因包括以下几个方面。首先，ResNet-18 本身已经通过 ImageNet 预训练获得了较强的特征表达能力，在 Oxford-IIIT Pet Dataset 上已经达到较高准确率，SE-block 的额外提升空间有限。其次，SE-block 引入了额外参数，可能增加了过拟合风险。第三，新增的 SE-block 参数无法直接从原始 ImageNet ResNet-18 中继承，只能随机初始化，因此需要更合适的学习率或训练策略才能充分发挥作用。第四，当前实验仍使用与 Baseline 相同的超参数，而带注意力模块的模型可能需要单独调参。

尽管 SE-block 未超过 Baseline，本实验仍完成了在 Baseline 模型基础上手动修改网络结构并引入注意力模块的要求，并通过实验比较了注意力模型和原始 Baseline 的分类准确率。

## 八、综合结果对比

最终三组核心实验结果如下：

| 实验 | 模型 | 预训练 | 注意力机制 | Epochs | Best Val Acc | Test Acc |
|---|---|---|---|---:|---:|---:|
| Baseline | ResNet-18 | 是 | 无 | 75 | 0.9334 | 0.8724 |
| 预训练消融 | ResNet-18 | 否 | 无 | 75 | 0.4959 | 0.4424 |
| 注意力机制 | ResNet-18 | 是 | SE-block | 75 | 0.9063 | 0.8621 |

从整体结果来看，ImageNet 预训练对模型性能影响最大。使用预训练权重后，ResNet-18 的测试准确率相比随机初始化模型提升了约 43 个百分点。这说明迁移学习在中小规模细粒度图像分类任务中非常有效。

超参数搜索表明，主干网络学习率不宜过大。较高的 backbone learning rate 会明显降低模型性能，说明微调预训练网络时需要谨慎更新已有特征。分类层由于是随机初始化的，可以使用更大的学习率。

注意力机制实验表明，SE-block 在当前设置下没有超过原始 Baseline。该结果提示我们，模型结构改进并不一定带来稳定提升，尤其是在 Baseline 已经较强、数据规模有限或注意力模块未单独调参的情况下。实验中仍可以观察到 SE-ResNet-18 保持了较高准确率，说明该结构能够正常适配任务，但若要进一步提升性能，可能需要针对 SE-block 重新搜索学习率、训练轮数、数据增强策略或正则化强度。

## 九、结论

本文完成了基于 ResNet-18 的 Oxford-IIIT Pet Dataset 宠物品种识别实验。首先，本文将 ImageNet 预训练 ResNet-18 的最后分类层替换为 37 类输出层，并采用分层学习率策略进行微调。实验结果表明，预训练 ResNet-18 能够在该任务上取得较好的分类性能。

其次，本文通过网格搜索分析了不同学习率组合对模型性能的影响。结果显示，主干网络学习率为 0.0001、分类层学习率为 0.001 时表现较好；过大的主干学习率会破坏预训练特征，导致泛化性能下降。

再次，本文进行了预训练消融实验。结果表明，在相同训练设置下，ImageNet 预训练模型显著优于随机初始化模型，说明预训练对中小规模细粒度分类任务具有明显帮助。

最后，本文在 ResNet-18 的 BasicBlock 中手动加入 SE-block 通道注意力机制，并与原始 Baseline 进行了对比。实验结果显示，SE-block 版本测试准确率略低于 Baseline，说明注意力机制在当前设置下没有带来额外提升。这一结果也说明，模型结构改进需要结合具体数据集、训练策略和超参数进行综合调优。

综上，本任务完整实现了数据集加载、预训练 CNN 微调、超参数分析、预训练消融实验以及注意力机制实验，满足任务要求。
